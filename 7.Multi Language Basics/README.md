# 🌐 Module 7: Multi-Language Basics (11 Languages)

As a software engineer, you will eventually work with multiple programming languages. The good news is: **programming logic (variables, loops, conditions) is universal.** Only the syntax changes.

This module provides a quick-reference guide and compilable/runnable basic scripts in **11 different languages** so you and your juniors can see how basic algorithms look across the industry.

---

## 📊 Syntax Reference Cheat Sheet

| Language | Console Output | Variable Declaration | If-Else Check | For Loop (1 to 5) |
| :--- | :--- | :--- | :--- | :--- |
| **C** | `printf("Hello\n");` | `int x = 10;` | `if(x > 5) {}` | `for(int i=1; i<=5; i++) {}` |
| **C++** | `std::cout << "Hello\n";` | `int x = 10;` | `if(x > 5) {}` | `for(int i=1; i<=5; i++) {}` |
| **Python** | `print("Hello")` | `x = 10` | `if x > 5:` | `for i in range(1, 6):` |
| **Java** | `System.out.println("Hello");` | `int x = 10;` | `if(x > 5) {}` | `for(int i=1; i<=5; i++) {}` |
| **Kotlin** | `println("Hello")` | `var x = 10` / `val` | `if(x > 5) {}` | `for(i in 1..5) {}` |
| **JavaScript**| `console.log("Hello");` | `let x = 10;` | `if(x > 5) {}` | `for(let i=1; i<=5; i++) {}` |
| **Go** | `fmt.Println("Hello")` | `x := 10` | `if x > 5 {}` | `for i := 1; i <= 5; i++ {}` |
| **C#** | `Console.WriteLine("Hello");` | `int x = 10;` | `if(x > 5) {}` | `for(int i=1; i<=5; i++) {}` |
| **Swift** | `print("Hello")` | `var x = 10` | `if x > 5 {}` | `for i in 1...5 {}` |
| **Rust** | `println!("Hello");` | `let x = 10;` | `if x > 5 {}` | `for i in 1..=5 {}` |
| **PHP** | `echo "Hello\n";` | `$x = 10;` | `if($x > 5) {}` | `for($i=1; $i<=5; $i++) {}` |
