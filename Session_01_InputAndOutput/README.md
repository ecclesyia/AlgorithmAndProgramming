# 📂 Session 01: Input and Output

This session covers the absolute fundamentals of programming: printing text to the console, declaring variables, and taking user inputs. 

To help you and your juniors widen your perspective, we have prepared equivalent Hello World and variables declaration scripts in **11 different programming languages**!

---

## 💡 Concepts Explained
1.  **Standard Output**: Printing text symbols to the screen so the user can read them.
2.  **Variables & Data Types**: Reserving named locations in the computer's memory (RAM) to store data (e.g. integers, floats, characters, and strings).
3.  **Standard Input**: Capturing keyboard keystrokes from the user and saving them into variables.

## 🚀 Preparing for What's Next: CS50 / MIT Context
*   **Why we learn this**: To write commands that let users communicate with computers. Without input and output, programs would be black boxes that do work but can't display results.
*   **What we use this for**: Creating user interfaces, reading keyboard commands, and print-debugging.
*   **Where to start next**: Learn how to manipulate these variables mathematically, which leads to **Arithmetic and Operators** in Session 02.
